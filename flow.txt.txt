Based on the requirements, we'll create **10 main pages** for the frontend:

## 📱 **Frontend Pages Overview**

### **Public Pages (3)**
1. **Landing/Home Page** (`/`)
   - Hero section with platform overview
   - Investment packages display (Apex1 & Apex2)
   - Features/benefits section
   - How it works
   - Call-to-action buttons (Login/Register)
   - Footer with company info

2. **Login Page** (`/login`)
   - Email/password form
   - Link to register
   - Forgot password option

3. **Register Page** (`/register`)
   - Registration form with email/password
   - Optional referral code field
   - Link to login

### **User Pages (6)**
4. **User Dashboard** (`/dashboard`)
   - Summary cards (total invested, total withdrawn, active investments)
   - Recent transactions
   - Bonus summaries (referral & retrading)
   - Quick action buttons
   - Notification area

5. **Investment Page** (`/investments`)
   - List of all user investments
   - Filter by status (active/pending/completed)
   - Each investment card shows:
     - Plan type
     - Amount invested
     - Expected return
     - Progress bar
     - Next withdrawal date
     - Withdrawal button (when available)

6. **Create Investment Page** (`/investments/create`)
   - Plan selection (Apex1/Apex2)
   - Amount input with min/max validation
   - Real-time return calculation
   - Payment instructions
   - Payment proof upload
   - Terms confirmation

7. **Profile/Bank Details Page** (`/profile`)
   - Personal information (editable)
   - Bank details form (locked after first save)
   - Account statistics
   - Referral code display

8. **Referrals Page** (`/referrals`)
   - Referral link with copy button
   - Total referrals count
   - List of referred users with:
     - Email/name
     - Investment amount
     - Bonus earned
     - Status
   - Referral bonus summary

9. **Bonuses Page** (`/bonuses`)
   - Tabbed view (Referral/Retrading)
   - Total available bonuses
   - Minimum withdrawal warning (10,000 NGN)
   - List of bonuses with status
   - Withdrawal button (when eligible)

### **Admin Pages (4)**
10. **Admin Dashboard** (`/admin`)
    - Statistics cards (users, investments, withdrawals)
    - Charts/graphs for platform activity
    - Recent activities

11. **Admin Users Page** (`/admin/users`)
    - List of all users
    - Search/filter functionality
    - User details modal
    - Block/Unblock actions

12. **Admin Investments Page** (`/admin/investments`)
    - Pending confirmations
    - Active investments
    - Payment proof review
    - Confirm/Decline actions

13. **Admin Settings Page** (`/admin/settings`)
    - Investment limits configuration
    - Bonus percentages adjustment
    - System toggle (enable/disable investments)
    - Company bank details
    - Withdrawal settings

### **Additional Pages (3)**
14. **Transactions Page** (`/transactions`)
    - Complete transaction history
    - Filter by type/status
    - Export functionality

15. **Withdrawal Page** (`/withdrawals`)
    - Available withdrawals
    - Withdrawal history
    - Bank details verification

16. **Support/Contact Page** (`/support`)
    - Contact form
    - FAQ section
    - Support information

## 🗺️ **Navigation Flow**

```
Landing Page
    ├── Login → Dashboard
    ├── Register → Dashboard
    └── View Packages → Create Investment

Dashboard
    ├── Investments → Investment List → Create/View Details
    ├── Profile → Edit Profile/Bank Details
    ├── Referrals → View/Analytics
    ├── Bonuses → View/Withdraw
    └── Transactions → History

Admin Area
    ├── Dashboard → Overview
    ├── Users → Manage Users
    ├── Investments → Confirm/Decline
    └── Settings → Configure Platform
```

This structure covers all requirements:
- ✅ User investment flows
- ✅ Referral system display
- ✅ Bonus tracking
- ✅ Bank details management
- ✅ Admin controls
- ✅ Withdrawal processes
